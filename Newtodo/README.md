# Todo List React Native avec Expo

Une application Todo List simple et élégante construite avec React Native et Expo SDK 54.

## Fonctionnalités

- ✅ Ajouter des tâches
- ✅ Cocher/décocher des tâches
- ✅ Supprimer des tâches
- ✅ Persistance des données avec AsyncStorage
- ✅ Interface simple et moderne (bleu/blanc)

## Installation

**Note pour Windows PowerShell :** Si vous rencontrez une erreur de politique d'exécution, utilisez `npm.cmd` au lieu de `npm` :

```powershell
npm.cmd install
```

Ou utilisez le script helper :
```powershell
.\use-npm.ps1 install
```

## Démarrage

```bash
npm start
```

Ou avec PowerShell :
```powershell
npm.cmd start
# ou
.\use-npm.ps1 start
```

Puis appuyez sur:
- `a` pour ouvrir sur Android
- `i` pour ouvrir sur iOS
- `w` pour ouvrir dans le navigateur web

## Technologies utilisées

- Expo SDK 54
- React 19.1.0
- React Native 0.76.5
- AsyncStorage pour la persistance
