# Todo List - React Native avec Expo

Une application Todo List simple et élégante avec persistance des données.

## Fonctionnalités

- ✅ Ajouter des tâches
- ✅ Cocher/décocher des tâches
- ✅ Supprimer des tâches
- ✅ Persistance des données avec AsyncStorage
- ✅ Interface simple et moderne (bleu/blanc)

## Installation

1. Installer les dépendances :
```bash
npm install
```

2. Démarrer l'application :
```bash
npm start
```

3. Scanner le QR code avec l'application Expo Go sur votre téléphone, ou appuyer sur :
   - `a` pour Android
   - `i` pour iOS
   - `w` pour Web

## Structure du projet

- `App.js` - Composant principal de l'application
- `package.json` - Dépendances du projet
- `app.json` - Configuration Expo

## Technologies utilisées

- React Native
- Expo
- AsyncStorage pour la persistance des données
